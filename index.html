xai/・openai/ で始まるモデル名で Grok / ChatGPT も使えます(例: xai/grok-4.3, openai/gpt-5.3-chat-latest)。対応する上のキーが必要です。

crosstalk へようこそ

LLM が動かす恋愛群像劇シミュレーションです。あなたは登場人物の一人に「乗り移り」、毎日の行動を自由に記述して物語を転がします。全員の秘密や鍵垢(非公開SNS)まで見えるので、誰の何を暴くかはあなた次第。

遊び方(進め方)

  1. ⚙ 設定で Anthropic APIキーを入力します(下記参照)。
  2. 行動計画タブで今日の行動を決めます(自分で書く / AIに考えてもらう / 🎲おまかせ)。
  3. 「この計画で一日を進める ▶」を押すと、その一日がシミュレートされます(1〜2分)。
  4. できごと・メッセージ・鍵垢で結果を確認。気になる相手に自分から DM を送ってもOK。
  5. これを繰り返して物語を進め、区切りがついたら終幕で総括を作れます。

タブの内容

  • あらすじ … 物語の舞台・状況・登場人物
  • メンバー … 各キャラの性格・秘密・目標・関係(クリックで詳細・前後移動・記憶の改変)
  • 行動計画 … 今日の行動を入力して一日を進める。「✋神の采配」で強制イベントも
  • できごと … その日に起きた出来事を小説で表示
  • メッセージ … キャラ同士のDM・グループチャット。あなたも送信できる
  • 鍵垢 … 各キャラの非公開SNS(誰にも見せない本音)
  • 終幕 … 全記録を開示するシーズン総括
  • LLMログ … 送信したプロンプト・応答・コスト

Anthropic APIキーの入手

console.anthropic.com でアカウントを作成し、「API Keys」からキーを発行します(sk-ant-…)。キーはこのブラウザにのみ保存され、Anthropic 以外には送信されません。トークン消費が大きいので、チャージ式(プリペイド残高)のキーを推奨します(残高が尽きれば自動で止まります)。

Anthropic 以外に OpenAI(ChatGPT)・xAI(Grok) も使えます。⚙設定で各キーを入れ、モデル名を openai/… ・ xai/… にするだけ。
